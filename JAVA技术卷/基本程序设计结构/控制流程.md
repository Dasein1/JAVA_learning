# 1.块作用域
块（即复合语句）是指由一对大括号括起来的若干条简单的Java语句。块确定了变量的作用域。一个块可以嵌套在另一个块中。
```java
public static void main(String[] args)
{
    int n;
    ...
    {
        int k;
    } // k is only defined up to here
}
```
但是，不能在嵌套的两个块中声明同名的变量。
```java
public static void main(String[] args)
{
    int n;
    ...
    {
        int k;
        int n; // Error can't redefine n in inner block
        ...
    }
}
```
