# 14 多线程编程
## 14.1 linux线程概述
### 14.1.1 线程模型
#### 线程的种类
线程分为内核线程和用户线程
内核线程运行在内核空间，由内核来调度
用户线程运行在用户空间，由线程库来调度，当进程的一个内核线程获得CPU的使用权，它就加载并运行一个用户线程。

### 线程的实现方式
* 完全在用户空间实现，M个用户空间线程对应1个内核线程
  * 线程库负责管理所有执行线程，比如线程的优先级、时间片，内核仍然把整个进程作为最小单位来调度的
* 完全由内核调用，1个用户空间线程被映射为1个内核线程
  * 创建、调度线程的任务都交给了内核，运行在用户空间的线程无需执行管理任务
* 双层调度，内核调度M个内核线程，线程库调度N个用户线程
