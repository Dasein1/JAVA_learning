# 得到一个数的每一位数字
## 1.直接将数字转换为字符串，然后转换为字符数组。
```java
int n=12345;
char[] chars=String.valueOf(n).toCharArray();
for (int j=0;j<chars.length;j++){
    System.out.print(chars[j]+' ');
}
```

## 2.末位取余法（从后往前依次得到一个数的每一位）
```java
int n=12345;
int[] a=new int[5];
for (int i=0;i<5;i++){
    a[i]=n%10;
    n=n/10;
}
for (int x:a){
    System.out.print(x+" ");
}
```
